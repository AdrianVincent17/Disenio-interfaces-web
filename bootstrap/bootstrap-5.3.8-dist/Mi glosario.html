Etiquetas HTML Bootstrap 5 Conceptos CSS

Etiquetas HTML Semánticas

Son etiquetas que describen su significado o propósito al navegador y al desarrollador.

<section>
Define una sección temática en un documento. Es un agrupador genérico con sentido.
<article>
Define contenido independiente y autocontenido (ej. un post de blog, un producto).
<nav>
Define un bloque de enlaces de navegación (menús).
<aside>
Define contenido relacionado pero separado del principal (ej. una barra lateral).

Chuleta de Bootstrap 5

Conceptos clave del framework de CSS.

Sistema de Rejilla (Grid)
Se basa en 12 columnas. Se usa con .container, .row, y .col-* (ej. .col-md-6).
.container vs .container-fluid
.container: Tiene un ancho máximo (max-width) que cambia según el tamaño de pantalla. Deja márgenes a los lados.
.container-fluid: Ocupa siempre el 100% del ancho (width: 100%).
Clases de Espaciado (Spacing)
Permiten añadir margin o padding rápidamente.
Formato: {propiedad}{lado}-{tamaño}
- Propiedad: m (margin), p (padding).
- Lado: t (top), b (bottom), s (start/left), e (end/right), x (eje X), y (eje Y).
- Ejemplo: .mt-3 (margin-top de tamaño 3), .px-5 (padding en X de tamaño 5).

Conceptos CSS

Definiciones clave de CSS moderno.

Flexbox
Modelo de diseño unidimensional. Se activa con display: flex;.
- Eje principal: justify-content (space-between, center, etc).
- Eje secundario: align-items (center, flex-start, etc).
CSS Grid
Modelo de diseño bidimensional (filas y columnas). Se activa con display: grid;.
Perfecto para layouts complejos de página.
