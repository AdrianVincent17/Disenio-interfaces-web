1. Introducción a Bootstrap

¿Qué es Bootstrap?
Un framework de diseño web (CSS y JS) creado por Twitter. Es Open Source y facilita la creación de sitios web con diseño adaptable ("responsive") y bajo la filosofía "Mobile First".
Ventajas
Permite crear diseños adaptables que se ajustan a cualquier pantalla (móvil, tablet, escritorio). Acelera el desarrollo, asegura consistencia visual y es compatible con todos los navegadores modernos.
Instalación Remota (CDN)
Es la forma más rápida y recomendada para prototipar. Se enlaza a los archivos de Bootstrap alojados en un servidor externo (CDN). Requiere conexión a internet.

Código para el <head> (CSS):

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet"
integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">

Código antes de cerrar </body> (JavaScript):

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"
integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script>
Instalación Local (Archivos)
Se descargan los archivos .css y .js compilados desde la web oficial de Bootstrap y se alojan en carpetas dentro de tu proyecto. No requiere conexión a internet para funcionar.

Código para el <head> (CSS):

<!-- La ruta './css/' depende de dónde guardes el archivo -->
<link rel="stylesheet" href="./css/bootstrap.min.css">

Código antes de cerrar </body> (JavaScript):

<!-- La ruta './js/' depende de dónde guardes el archivo -->
<script src="./js/bootstrap.bundle.min.js"></script>

2. Sistema Grid (Rejilla) y Flexbox

Concepto Principal del Grid
Un sistema de rejilla "Mobile First" basado en 12 columnas. Permite estructurar el contenido de forma flexible y responsiva.
Jerarquía: .container, .row, .col-*
Clases de Columnas Responsivas
Define el ancho de la columna (1 a 12). Se pueden combinar breakpoints para diseños avanzados.
  • .col: Ancho automático. Reparto equitativo.
  • .col-6: Ocupa la mitad (6 de 12) en todos los tamaños.
  • .col-md-4: Ocupa 4 de 12 a partir del tamaño mediano md.
Alineación Horizontal de la Fila (Eje X)
Estas clases se aplican a la fila .row para distribuir las columnas a lo largo del eje principal. Se utiliza la utilidad justify-content-*:
  • .justify-content-start: Columnas a la izquierda.
  • .justify-content-center: Centra las columnas.
  • .justify-content-end: Columnas a la derecha.
  • .justify-content-around: Distribuye el espacio alrededor de cada columna.
  • .justify-content-between: Distribuye el espacio entre columnas (extremos pegados).
Ejemplo: `<div class="row justify-content-center">...</div>`
Alineación Vertical de la Fila (Eje Y)
Estas clases se aplican a la fila .row para alinear todas las columnas verticalmente dentro de la fila. Se utiliza la utilidad align-items-*:
  • .align-items-start: Alinea todas las columnas arriba (inicio del contenedor).
  • .align-items-center: Centra todas las columnas verticalmente.
  • .align-items-end: Alinea todas las columnas abajo (fin del contenedor).
  • .align-items-baseline: Alinea por la línea base del texto.
  • .align-items-stretch: Estira las columnas para que ocupen todo el alto (por defecto).
Ejemplo: `<div class="row align-items-center">...</div>`
Alineación Vertical Individual align-self-*
Esta clase se aplica a una columna individual .col-* para anular la alineación vertical de la fila y posicionarla de manera independiente. Se utiliza la utilidad align-self-*:
  • .align-self-start: Columna individual arriba.
  • .align-self-center: Columna individual centrada.
  • .align-self-end: Columna individual abajo.
Ejemplo: `<div class="col-4 align-self-end">...</div>` (Esa columna se va al fondo).
Gutters (Espaciado entre columnas)
Controla el espacio entre columnas (padding y margen negativo en la fila). Se aplica a la clase .row:
  • g-0: Elimina todos los gutters.
  • g-{tamaño}: Asigna gutter horizontal y vertical (tamaño **0-5**).
Márgenes y Rellenos (Spacing)
Puedes añadir márgenes (margin) o rellenos (padding) a cualquier elemento.
Formato: {propiedad}{lado}-{tamaño}
  • Propiedad: m (para margin), p (para padding).
  • Lado: t (top), b (bottom), s (start/left), e (end/right), x (ejes X), y (ejes Y).
  • Tamaño: 0 (cero), 1 (pequeño) a 5 (grande), y auto.
Ejemplos: .mt-3 (margin-top), .px-4 (padding horizontal), .mb-5 (margin-bottom), .mx-auto (centrar horizontalmente).

3. Textos y Tipografía

Display Headings
Encabezados más grandes y destacados.
Clases: .display-1, .display-2, ..., .display-6.
.lead
Clase para párrafos introductorios o destacados. Les da un tamaño de fuente mayor.
Alineación de Texto
.text-start, .text-center, .text-end.
**Responsivo:** .text-md-center (centra solo a partir de md).
Transformación de Texto
.text-lowercase, .text-uppercase, .text-capitalize.
Peso y Estilo (Font Weight/Style)
.fw-bold (negrita), .fw-normal, .fw-light.
.fst-italic (cursiva).
Blockquote (Citas)
.blockquote para la etiqueta <blockquote>.
.blockquote-footer: Para indicar la fuente de la cita.
Resaltar Código
<code>: Para fragmentos de código en línea.
<pre>: Para bloques de código de varias líneas.
<kbd>: Para indicar atajos de teclado.

4. Imágenes y Tablas

Imágenes Fluidas (Responsivas)
.img-fluid: La clase más importante. Aplica max-width: 100%; y height: auto;.
Estilos de Imagen
.rounded (esquinas redondeadas).
.rounded-circle (círculo).
.img-thumbnail (borde de miniatura).
Alinear Imágenes
.float-start (izquierda), .float-end (derecha).
Centrar: .d-block y .mx-auto.
Tablas (Base)
.table: Es la clase base obligatoria para <table>.
Estilos de Tabla
.table-striped: Filas con colores alternos.
.table-bordered: Con bordes.
.table-hover: Efecto al pasar el ratón.
.table-sm: Tabla más compacta.
Tablas Oscuras / Claras
Bootstrap 5 adapta las tablas automáticamente al modo oscuro. .table-dark sigue forzando el modo oscuro si fuera necesario.
Tablas Responsivas
.table-responsive: Se usa en un <div> que envuelve a la <table>. Añade scroll horizontal en pantallas pequeñas si la tabla no cabe.

5. Colores y Fondos

Colores Temáticos
Bootstrap utiliza nombres semánticos para los colores que pueden cambiar con el tema.
  • primary (Azul, principal)
  • secondary (Gris, secundario)
  • success (Verde, éxito)
  • danger (Rojo, peligro)
  • warning (Amarillo, advertencia)
  • info (Celeste, información)
  • light (Blanco/Gris claro)
  • dark (Negro/Gris oscuro)
  • white (Blanco puro)
  • black (Negro puro)
Clases de Texto (Text Colors)
Para cambiar el color de la fuente: .text-{color}.
Ejemplos: .text-primary, .text-success, .text-white, .text-muted (texto grisáceo). 
<p class="text-danger">¡Cuidado!</p>
Clases de Fondo (Background Colors)
Para cambiar el color de fondo: .bg-{color}.
Ejemplos: .bg-primary, .bg-dark, .bg-warning. 
<div class="p-3 bg-success text-white">Fondo Verde</div>
Opacidad del Fondo (Background Opacity)
Puedes controlar la opacidad del color de fondo con la utilidad .bg-opacity-{porcentaje}.
Porcentajes: 10, 25, 50, 75, 100. 
<div class="p-3 bg-primary bg-opacity-50">50% de Opacidad</div>
Modo de Color (Dark/Light Mode)
Bootstrap 5.3+ admite la personalización del modo de color con el atributo data-bs-theme en la etiqueta <html> o <body>.
  • Modo Oscuro: <html data-bs-theme="dark">
  • Modo Claro: <html data-bs-theme="light">

4. Botones y Dropdowns

Clase Base de Botón
Todos los botones deben llevar la clase base .btn. 
<button class="btn">Botón Base</button>
Estilos de Botón
Define el color y estilo con .btn-{color}.
Ejemplos: .btn-primary, .btn-success, .btn-link (parece un enlace). 
<a href="#" class="btn btn-warning">Alerta</a>
Botones de Contorno (Outline)
Botones transparentes con borde de color: .btn-outline-{color}.
Ejemplo: .btn-outline-secondary.
Tamaño de Botón
Clases para cambiar el tamaño: .btn-sm (pequeño), .btn-lg (grande).
Botones de Bloque (Full Width)
Para que un botón ocupe todo el ancho disponible, se usa una utilidad de Display Flex: 
<div class="d-grid">
    <button class="btn btn-primary">Ocupar todo el ancho</button>
</div>
Estructura de Dropdown
Un menú desplegable. Se necesitan las clases .dropdown (en el contenedor padre), .dropdown-toggle (en el botón) y .dropdown-menu (en la lista).
  • Requiere el plugin JavaScript de Bootstrap para funcionar.
  • El botón debe tener data-bs-toggle="dropdown".
<div class="dropdown">
    <button class="btn btn-primary dropdown-toggle" data-bs-toggle="dropdown">Menú</button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">Opción 1</a></li>
        <li><a class="dropdown-item" href="#">Opción 2</a></li>
    </ul>
</div>
Elementos del Dropdown
Dentro de .dropdown-menu:
  • .dropdown-item: Elemento normal.
  • .dropdown-divider: Línea de separación (etiqueta <hr>).
  • .dropdown-header: Título de sección (etiqueta <h6>).

5. Navbars (Barras de Navegación)

Estructura Básica
La estructura estándar envuelve la navegación en la etiqueta <nav> con la clase .navbar, un contenedor .container-fluid y la lista de enlaces .navbar-nav. 
<nav class="navbar navbar-expand-lg bg-dark">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">Mi Marca</a>
        ... Menú ...
    </div>
</nav>
Expansión (Breakpoints)
La clase .navbar-expand-{breakpoint} define a partir de qué tamaño la barra se colapsa en un botón de "hamburguesa".
  • .navbar-expand-lg: Se colapsa en pantallas pequeñas y medianas, se expande a partir de `lg`.
  • .navbar-expand-none: Nunca se colapsa.
Clases de Color de Navbar
Define el color de fondo y el esquema de color del texto:
  • Fondo: .bg-{color} (ej. .bg-primary, .bg-dark).
  • Tema: .navbar-dark (Texto claro para fondos oscuros) o .navbar-light (Texto oscuro para fondos claros).
Toggle (Botón de Menú Colapsado)
Es el botón de "hamburguesa" que aparece en pantallas pequeñas.
  • Clase: .navbar-toggler.
  • Debe apuntar al contenido colapsable con data-bs-target (usando el ID del contenido).
  • Icono: <span class="navbar-toggler-icon"></span>.
Formularios y Búsqueda
Se pueden insertar formularios de búsqueda dentro de la navbar.
Clase: .d-flex (utilidad Flexbox para alinear elementos horizontalmente). 
<form class="d-flex" role="search">...</form>
